什么是 XAMPP?

XAMPP 是最流行的 PHP 开发环境

XAMPP 是完全免费且易于安装的 Apache 发行版，其中包含 MySQL、PHP和Perl。XAMPP 开放源码包的设置让安装和使用出奇容易。

下载地址：https://www.apachefriends.org/download.html

进入 Manage Servers，打开 MySQL Database 和 Apache Web Server
