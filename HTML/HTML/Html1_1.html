什么是 HTML?

HTML 是用来描述网页的一种语言

HTML 指超文本标记语言(Hyper Text Markup Language)

HTML 不是编程语言，是一种标记语言

HTML5 的新特性:

用于绘画的 canvas 标签

用于媒介回放的 video 和 audio 元素

对本地离线存储的更好支持

新的特殊内容元素

如: article、footer、header、nav、section

新的表单控件

如: calendar、date、time、email、url、search
