JavaScript

JavaScript 介绍

  1. JavaScript 是互联网上最流行的脚本语言，这门语言可用于 web 和 HTML，更可广泛用于服务器、pc端、移动端。
  2. JavaScript 脚本语言:

    JavaScript 是一种轻量级的编程语言

    JavaScript 是可插入 HTML 页面的编程代码

    JavaScript 插入 HTML 页面后，可由所有的浏览器执行

  3. JavaScript 用法:

    HTML 中的脚本必须位于 <script></script> 标签之间

    脚本可被放置在 HTML 页面的 <body>和 <head> 部分中

  4. JavaScript 使用限制:

    在 HTML 中，不限制脚本数量

    通常会把脚本放置于 <head> 标签中，以不干扰页面内容。

  5. 标识符:

    JavaScript 标识符必须以字母、下划线或美元符号开始

  6. 外部引用脚本:

    <script src="X.js"></script>

运算符

三个等号

三个等号(=== 或者 !==)必须满足类型也要相同

var i = "10";

var j = 10;

document.write(i===j); 结果为: false

document.write(i==j); 结果为: true

三目运算符

document.write(i>10?"i大于10":"i小于10");
