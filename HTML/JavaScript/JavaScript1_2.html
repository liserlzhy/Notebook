JavaScript 异常捕获

介绍

  1. 异常:

    当 JavaScript 引擎执行 JavaScript 代码时，发生了错误，导致程序停止运行

  2. 异常抛出:

    当异常产生，并且将这个异常生成一个错误信息

  3. 异常捕获: 
    try{
    发生异常的代码块;
}catch(err){
    错误的信息处理;
}

实例

<script>
    function demo() {
        try{
            var e = document.getElementById("txt").value;
            if(e==""){
                throw "空异常";
            }
        }catch (err){
            alert(err);
        }
    }
</script>
<form>
    <input title="用户名" id="txt" type="text">
    <input id="bt" type="button" onclick="demo()" value="确认"/>
</form>
