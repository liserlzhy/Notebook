SQL LIKE 操作符

LIKE 操作符用于在 WHERE 字句中搜索列中指定模式。

SQL Server 的语法:

SELECT column_name(s)
FROM table_name
WHERE column_name LIKE pattern

实例1：

列出以 "N" 开始的城市: 

SELECT * FROM Persons
WHERE City LIKE 'N%'

注释: "%" 可用于定义通配符（模式中缺少的字母）。其他形式 "%N"、"%N%"

实例2:

通过使用 NOT 关键字，选取不包含 "lon" 的城市的人:

SELECT * FROM Persons
WHERE City NOT LIKE '%lon%'
