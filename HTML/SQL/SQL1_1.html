SQL 基础命令

操作 例子
创建数据库： CREATE DATABASE db_name;
创建表格： CREATE TABLE tb_name(tr_name1 data_type , tr_name2 data_type);
插入数据： INSERT INTO tb_name {列名，列名}VALUES(1_1,1_2),(2_1,2_2);
显示数据库： SHOW DATABASES;
显示表格格式： SHOW COLUMNS FROM tb_name;
显示表格内容： SELECT * FROM tb_name;
显示特定列： SELECT 列名 FROM tb_name;
筛选数据： SELECT * FROM tb_name WHERE tr_name1='#' AND tr_age>100;
排序： SELECT Company, OrderNumber FROM Orders ORDER BY Company, OrderNumber;
逆序： SELECT Company, OrderNumber FROM Orders ORDER BY Company DESC, OrderNumber ASC;
更新： UPDATE Person SET Address = 'Zhongshan 23', City = 'Nanjing' WHERE LastName = 'Wilson';
删除表中的行 DELETE FROM 表名称 WHERE 列名称 = 值
删除所有行 DELETE FROM table_name 或者 DELETE * FROM table_name
