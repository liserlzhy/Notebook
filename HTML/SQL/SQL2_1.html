SQL TOP字句

Top子句用于规定要返回的记录的数目。

注释: 并非所有的数据库系统都支持 TOP 子句。MySQL不支持，可用limit替代

SQL Server 的语法:

SELECT TOP number|percent column_name(s)
FROM table_name

实例：

SELECT TOP 2 * FROM Persons
SELECT TOP 50 PERCENT * FROM Persons
